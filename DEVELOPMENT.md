## Requirements

Please install:

- **Node.js** (v16 or later recommended)

- **npm** (comes with Node)  
  *(or Yarn / pnpm if preferred)*

---
## 1. Install Dependencies
Navigate to the project folder
```bash
cd roam.io/frontend
```

Inside the project folder, run:

```bash
npm install
```

## 2.  Run Development Server

```bash
npm run start
```
